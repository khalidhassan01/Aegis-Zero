The 3 Puppets — Each Has One Job
🔭
Scout
aegis-fast · always
Gathers context before reasoning begins. Searches episodic memory, checks the semantic cache, retrieves relevant documents. Produces a structured briefing for Forge.
"What does Forge need to know to answer this well?"
⚒
Forge
aegis-deep (complex) · aegis-fast (simple)
The reasoning and generation engine. Receives Scout's briefing and a 4-layer context-engineered prompt. Produces the actual response. Can run in parallel (×2–3) for research tasks.
"What is the best possible response to this request?"
🔎
Auditor
aegis-fast · complex + code only
Reviews Forge's output before delivery. Checks: factual errors, incomplete answers, security leaks, code bugs, missed requirements. Returns approved response or revised version.
"Is this correct and safe to send?"
