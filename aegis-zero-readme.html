What is Aegis Zero?

Aegis Zero is a complete self-hosted AI infrastructure blueprint — designed, architected, and refined over 7 versions to achieve something most people think requires paid cloud services, enterprise budgets, or DevOps teams:

A fully private, production-grade AI agent system that costs nothing to run. Forever.

It runs on Oracle Cloud's Always Free ARM instance — 4 OCPUs, 24GB RAM, dedicated compute, Frankfurt region (GDPR by default). The server is invisible to the internet. All data stays in your instance. All AI runs locally. The system improves itself every night while you sleep.

Built on top of Hermes Agent as the AI core — Aegis Zero is the complete production infrastructure wrapped around it: the security architecture, the LLM routing engine, the RAG memory system, the API fallback pools, the deployment strategy, and the self-improvement framework.

The 5 Laws

These aren't configuration choices. They're the principles every architectural decision was made against.

LAW 1Security IS the foundation, not a feature added later. Every component is both functional AND a security boundary.
LAW 2Everything self-hosted = everything private. No personal data leaves the instance except by explicit choice.
LAW 3Every component serves multiple purposes. No dead weight. Nothing installed that doesn't earn its place.
LAW 4Every layer is replaceable without breaking others. Modular by design.
LAW 5The system gets smarter autonomously. You use it — it improves. No extra effort from you ever.

Security Architecture — 6 Layers

Security is not added on top. It IS the structure. Every component is both functional AND a security boundary.

0
Tailscale Zero-Port Mesh
Server invisible to the internet
🔴 Most Impactful
Two commands. Server becomes invisible. Zero ports open to the internet.

Before: Ports 443+2222 open → attack surface exists → scanners find you
After: Zero public ports → instance invisible → attackers cannot find what they cannot see
1
Oracle VCN Firewall
Deny ALL inbound — zero exceptions
🔴 Critical
After Tailscale: Security List = deny ALL inbound. Egress: HTTPS 443 outbound only. Reserved Static IP never changes after reboot.
2
OS Hardening
iptables DROP · Fail2ban · Metadata IP block
🔴 Critical
iptables default DROP policy. Fail2ban brute force protection. Oracle metadata endpoint blocked at 169.254.169.254 — prevents credential exfiltration. Unattended security upgrades enabled.
3
Nginx Reverse Proxy
Internal routing only
🟡 High
Internal routing only. No external binding. All services communicate on localhost. SSL termination inside the mesh.
4
Application Security
Command approval · PII redaction · Secret scanner
🟡 High
Command approval before execution. PII redaction on external API calls. Secret scanner on all outputs. Inline diff previews before any file changes. Stealth browser for research agents.
5
Session Security
No reverse access · Session continuity IDs
🟡 High
No reverse access path to client devices exists. Session continuity IDs across all interfaces. Credential pool isolation per provider.
